# Skills

## Install

```bash
npx skills add o2-exchange/skills
```

Skills are organized by domain.

- **[o2-reference](./o2-reference)** — Shared O2 API/session/signing reference docs.
- **[o2-sdk](./o2-sdk)** — TypeScript and Rust O2 SDK usage.
- **[fast-bridge](./fast-bridge)** — Fast-bridge deposit and withdrawal skills with TypeScript and Rust reference flows.
